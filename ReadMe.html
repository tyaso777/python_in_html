Pyodide Data Sandbox - User Guide

1. Overview

This tool runs Python directly in your browser using Pyodide. You can upload data files, write Python code with pandas, numpy, and matplotlib, and then inspect the output, generated images, and downloadable result files.

Main features:

  • No local Python installation required
  • Supports CSV, JSON, and TXT files
  • Editable Python code area
  • Standard output and error display
  • Image preview for generated plots
  • Download links for generated output files

2. Basic Workflow

  1. Upload one or more files
  2. Write or edit Python code
  3. Click Run Python
  4. Check output, images, and downloadable files

3. Uploading Files

Files can be uploaded either by drag and drop or by using the file selection button.

Uploaded files are stored in the browser-side virtual file system and can be accessed from Python under:

/uploads/your_file_name

Example:

df = pd.read_csv("/uploads/sample_user.csv")
File names must match exactly. For example, sample_user.csv and sample-user.csv are different names.

4. Writing Python Code

You can freely edit the Python code in the editor area. The environment supports common data-processing tasks with:

  • pandas
  • numpy
  • matplotlib

Simple example:

import pandas as pd

df = pd.read_csv("/uploads/sample_user.csv")
print(df.head())

5. Standard Output and Errors

Anything printed with print() appears in the Stdout / stderr area.

If an error occurs, the traceback is also shown there.

Example:

print("Hello")
print(df.shape)

6. Saving Images

To show a plot in the Images area, save it under /outputs/. 

import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [10, 20, 15])
plt.title("Sample Plot")
plt.savefig("/outputs/plot.png")

Saved image files such as .png are automatically shown in the UI.

7. Saving Downloadable Files

Files written under /outputs/ appear in the Downloads area. 

df.to_csv("/outputs/result.csv", index=False)

This can be used for processed CSV files, JSON files, text files, and plot images.

8. Sample Use Case: Join Two CSV Files and Draw a Chart

Upload the following two files:

  • sample_user.csv
  • sample_purchase.csv

Example code:

import pandas as pd
import numpy as np
import matplotlib
matplotlib.use("Agg")
import matplotlib.pyplot as plt

users = pd.read_csv("/uploads/sample_user.csv")
purchases = pd.read_csv("/uploads/sample_purchase.csv")

df = pd.merge(purchases, users, on="user_id", how="left")

summary = (
    df.groupby("user_name")["amount"]
    .sum()
    .reset_index()
    .sort_values("amount", ascending=False)
)

print(summary)

plt.figure(figsize=(8, 4))
plt.bar(summary["user_name"], summary["amount"])
plt.title("Total Purchase Amount by User")
plt.xlabel("User")
plt.ylabel("Amount")
plt.xticks(rotation=30)
plt.tight_layout()
plt.savefig("/outputs/user_total.png", dpi=140)

summary.to_csv("/outputs/user_summary.csv", index=False)

9. How to Check Uploaded File Names

If you are unsure about the uploaded file names, run: 

import os
print(os.listdir("/uploads"))

This is useful when you get a FileNotFoundError.

11. Notes and Limitations

All processing happens in the browser. Uploaded files are not automatically sent to a server by this tool.
  • Large files may be slow because processing uses browser memory
  • Uploaded files disappear when the page is reloaded
  • Only files under /uploads/ can be read
  • Only files under /outputs/ appear as downloadable results

12. Troubleshooting

FileNotFoundError

The file path in Python does not match the uploaded file name. 

import os
print(os.listdir("/uploads"))

No image appears

Make sure the plot is saved under /outputs/. 

plt.savefig("/outputs/plot.png")

No downloadable file appears

Make sure the file is written under /outputs/.

13. Minimal Template

If you only need a simple CSV read and export workflow, start with this: 

import pandas as pd

df = pd.read_csv("/uploads/your_file.csv")
print(df.head())

df.to_csv("/outputs/result.csv", index=False)
